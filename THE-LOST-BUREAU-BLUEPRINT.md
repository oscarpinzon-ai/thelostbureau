## Part 3: Production Workflow

### Script Generation

**Process:**
1. Use Claude Code to generate script templates (given video idea + keywords)
2. Manual research: 3-5 sources per video (Wikipedia, historical archives, Reddit)
3. Write 2,000-2,500 word script in English (conversational, not academic)
4. Include timestamps for key claims (build credibility)
5. Add call-to-action: "Let me know in the comments what you think..."

**Script Structure (Exact):**
- **Intro (0:00-0:30):** Hook + intro text (animated)
- **Body (0:30-27:00):** Main content in 2-3 min segments with B-roll transitions
- **Compelling segments (every 5-7 min):** Restate main claim, add detail
- **Build to climax (25:00-27:00):** "Why they're hiding this" reveal
- **Outro (27:00-28:00):** Subscribe + related video suggest

**Tone:** Authoritative but conversational (like a historian explaining at dinner)

### Thumbnail Generation (Nano Banana API)

**Batch Process:**
1. Create prompt template for each video (title + keywords + color palette)
2. Use Nano Banana API to generate 3-5 thumbnail variations
3. A/B test first 48 hours, pick winner
4. Store all variations in `/production/thumbnails/`

**Nano Banana Prompt Template:**
```
Generate a YouTube thumbnail in THE LOST BUREAU style:
- Dark charcoal background (#2D2D2D)
- Bold text: "[VIDEO TITLE]" in white/gold
- Include vintage archive/government imagery
- Small "THE LOST BUREAU" logo bottom-left
- Dimensions: 1280x720px
- Style: Mysterious, 1960s government aesthetic
```

### Voiceover (ElevenLabs API)

**Voices to Test:**
- **American Male (Neutral):** Neutral Male (US) — authoritative
- **Alternative:** Older Male (deeper voice) — more credible
- **British Option:** British Male — alternative appeal

**Settings:**
- Stability: 0.7 (clear, professional)
- Similarity Boost: 0.75 (natural tone)
- Style: Normal

**Process:**
1. Generate voiceover via ElevenLabs API (batch mode)
2. Download MP3 (save to `/production/voiceovers/`)
3. Import to DaVinci Resolve or CapCut
4. Sync with B-roll footage

### Video Assets (Veo3 + Stock Footage)

**B-Roll Sources (Priority):**
1. **Archive.org** (historical footage, free, public domain)
2. **Pexels** (stock photos, free)
3. **Unsplash** (modern imagery, free)
4. **Pixabay** (diverse assets, free)
5. **Veo3** (AI-generated sequences for transitions/fills)

**Asset Workflow:**
1. Create manifest: `assets-manifest.json` (video ID → all sources used)
2. Download footage for each video
3. Tag with timecodes for sync
4. Store in `/production/raw-footage/`

### Video Editing

**Tools:** CapCut (primary) or DaVinci Resolve (advanced color grading)

**Editing Guidelines:**
- Cut transitions: Every 2-3 seconds (fast pacing)
- Ken Burns effect: Slow zoom on static images (cinematic feel)
- Color grading: Desaturate -15%, warm color cast (nostalgic tone)
- Text overlays: Key claims at 3-5 second intervals
- Sound design: Subtle background music (low volume), no distracting SFX
- Final length: 26-34 minutes (optimized for 2-3 mid-roll ads)

**Quality checklist:**
- ✅ Audio levels normalized (-3dB peak)
- ✅ No dead time (every shot contributes)
- ✅ Text legible (40pt minimum)
- ✅ Color consistent throughout
- ✅ No harsh transitions

### Publishing Workflow

**Before Upload:**
1. Render final video (1080p60 or 4K, H.264)
2. Create SEO metadata (title, description, tags, category)
3. Schedule in YouTube Studio (pre-approved times)
4. Set thumbnail from pre-generated batch
5. Add cards/end screen (suggest 2 related videos)

**YouTube Optimization:**
- **Category:** Documentary / Education
- **Tags:** 8-12 tags (mix high-volume + niche keywords)
- **Description:** 300 words (includes 3-4 linked sources)
- **Hashtags:** 2-3 in description
- **Community post:** 24h before (tease video)

## Part 4: Technical Setup & Automation

### Claude Code Scripts (4 Core Utilities)

#### 1. Script Generator
**File:** `automation/claude-script-generator.js`

**Purpose:** Generate video scripts from video idea + keywords

**Input:** `{ videoTitle, keywords[], duration, tone }`

**Output:** Markdown script (ready for voiceover)

**Example Usage:**
```bash
node claude-script-generator.js \
  --title "The Government Doesn't Want You Remembering This Illegal Survival Skill" \
  --keywords "survival,emergency,food,banned" \
  --duration 28
```

#### 2. Thumbnail Batch Generator
**File:** `automation/thumbnail-batch-generator.js`

**Purpose:** Generate multiple thumbnail variations via Nano Banana API

**Input:** `{ videoTitle, keywords[], colorPalette, count: 3-5 }`

**Output:** 3-5 PNG files saved to `/production/thumbnails/[video-id]/`

**Config Required:**
```json
{
  "NANO_BANANA_API_KEY": "YOUR_KEY",
  "NANO_BANANA_MODEL": "stable-diffusion-xl",
  "OUTPUT_PATH": "./production/thumbnails"
}
```

#### 3. ElevenLabs Batch TTS
**File:** `automation/elevenlabs-batch-tts.js`

**Purpose:** Convert script text to voiceover MP3

**Input:** Script markdown file + voice selection

**Output:** MP3 file saved to `/production/voiceovers/[video-id]/`

**Config Required:**
```json
{
  "ELEVENLABS_API_KEY": "YOUR_KEY",
  "VOICE_ID": "ElevenLabs-voice-uuid",
  "OUTPUT_PATH": "./production/voiceovers"
}
```

#### 4. YouTube Scheduler
**File:** `automation/youtube-scheduler.js`

**Purpose:** Schedule uploads + track performance

**Input:** Video metadata + publish time

**Output:** Scheduled video link + performance tracker update

**Config Required:**
```json
{
  "YOUTUBE_API_KEY": "YOUR_KEY",
  "CHANNEL_ID": "YOUR_CHANNEL_ID",
  "SCHEDULE_DB": "./analytics/upload-schedule.json"
}
```

### API Keys & Secrets

**Required Services:**
| Service | Key | Purpose |
|---------|-----|---------|
| ElevenLabs | `ELEVENLABS_API_KEY` | Voiceover generation |
| Nano Banana | `NANO_BANANA_API_KEY` | Thumbnail generation |
| YouTube | `YOUTUBE_API_KEY` | Scheduling + analytics |
| GitHub | `GITHUB_TOKEN` | Auto-commit scripts |

**Storage:** `.env.local` (never commit to Git)
