## Scheduling Best Practices

### Pre-Upload (48 hours before)
- [ ] Video fully edited and rendered
- [ ] Thumbnail batch generated and A/B tested
- [ ] Metadata written (title, description, tags)
- [ ] SEO keywords verified
- [ ] Thumbnail dimensions confirmed (1280x720)
- [ ] Schedule in YouTube Studio as "Scheduled"

### 24 Hours Before Upload
- [ ] Create community post (tease video)
- [ ] Write compelling description
- [ ] Add timestamps if applicable
- [ ] Set custom thumbnail
- [ ] Add related video cards

### Upload Time (6 PM UTC)
- [ ] Video goes live
- [ ] Monitor first hour: CTR, comments
- [ ] Pin helpful comment
- [ ] Prepare Twitter/social snippets

### Post-Upload Monitoring (First 48 Hours)
- [ ] Track CTR (target: 6-8%)
- [ ] Monitor watch duration (target: 50%+)
- [ ] Note trending comments
- [ ] Check for technical issues (audio sync, buffering)
- [ ] Respond to early comments (builds engagement)

## Optimization Based on Performance

### If Video Underperforms (CTR < 5%)
1. Check thumbnail (A/B test winner from batch)
2. Revise title (test alternative version)
3. Check description SEO (missing keywords?)
4. Monitor watch duration (did script fail early?)

### If Video Overperforms (CTR > 8%)
1. Document title pattern (exact structure for next video)
2. Save winning thumbnail variation
3. Note keywords that drove traffic
4. Create similar video within 1-2 weeks

### If Watch Duration < 40%
1. Check first 3 minutes (hook too slow?)
2. Review pacing (cuts too fast/slow?)
3. Verify audio sync (voiceover out of sync?)
4. Analyze viewer comments (what did they want more of?)
